### Data Structure: 
+ StaticQueue
### Storage Method: 
+ Dynamic Array
### Key Methods:
+ Enqueue()  
	+ Complexity: O(1)
+ Dequeue()
	+ Complexity: O(1)
+ IsEmpty()
	+ Complexity: O(1)
+ Peek()
	+ Complexity: O(1)
+ IsFull()
	+ Complexity: O(1)
+ Size()
	+ Complexity: O(1)
+ Capacity()
	+ Complexity: O(1)

### Description: 
+ It is a data structure container that 
removes items based on the FIFO 
(First-In-First-Out) principle.
